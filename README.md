## Deep Vein Thrombosis (DVT) Blood Clot Detection

This project aims to develop an AI system for detecting Deep Vein Thrombosis (DVT) using ultrasound (DICOM) imaging. We request community support by contributing medical ultrasound data to improve model accuracy.

## Data Needed

- Ultrasound DICOM files related to Deep Vein Thrombosis  
- Short-axis and long-axis vein scans  
- Compression and non-compression views  
- Optional clinical notes

These contributions will support the development of accurate AI-based detection systems for DVT.

---

## Purpose

The goal of this project is to build a clean, research-quality dataset and machine learning models to assist in early and accurate detection of blood clots.
